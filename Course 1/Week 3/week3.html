More On Arrays

array.sort() – sorting.

array.reverse() – reverse.

array.sort().reverse() – descending order.

array.indexOf() – return number of position of element, or -1 if element not found.

array.indexOf(target, startPosition) – find from position.

array.lastIndexOf() – finding element backwards.

array2 = array.slice(1) – extract part of array from specific position. The method doesn't remove elemet from original array!

array2 = array.slice(startPosition, endPosition) – extract part of array from start to end position.

array.splice(position, quantity) – remove elements from array.

array.splice(position, 0, element) – adding element to array.

array.splice(position, quantity, elements) – replace elements in array.
