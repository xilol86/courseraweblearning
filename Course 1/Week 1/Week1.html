1. The basics of HTML

About HTML

HTML is main language for building html-pages.

Main idea of HTML using:

  1. The browser sends request to the server asking for a web page
  2. The server sends the web page back to the browser
  3. The browser displyas the web page

Idea of single page application (SPA):

  1. The browser sends request to the server asking for a small piece fo data
  2. The server sends a small piece fo data back to the browser
  3. The browser displyas the data (page not reloaded)

Alternative for HTML is SVG.
